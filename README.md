# NLPK
# NHS Digital Data Science Internship Project November 2019

## Run

Install the required libraries and run NLP.py with python3

```bash
conda env create -n NLP -f environment.yml
```
```bash
conda activate NLP
```

```bash
python3 NLP.py
```
